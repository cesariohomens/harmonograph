# Harmonograph Simulator

A web-based simulator for visualizing the patterns of a harmonograph using HTML, CSS, and JavaScript.

Demo: https://www.cesariogarcia.com/harmonograph

## Features

- Interactive controls for adjusting the frequency and damping of up to three pendulums.
- Real-time drawing mode.
- Color customization for both the drawing and the background.
- Responsive design for various screen sizes.

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/cesariohomens/harmonograph.git

2. Open index.html in your preferred web browser.
3. Use the provided controls to adjust the harmonograph's parameters and visualize the patterns.
